## Option B: Local Backend

Run the CVA backend on your own machine for:
- Offline use
- Data privacy
- Custom model configuration

### Requirements
- Python 3.10+
- OpenAI/Anthropic API keys

### Setup
1. Clone the CVA backend repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set your API keys in environment
4. Use **CVA: Select Backend Path** to point to the folder
