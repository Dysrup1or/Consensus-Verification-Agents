# Kling AI Video Ad Prompt - CVA (Consensus Verifier Agent)

> **Duration:** 15 seconds  
> **Purpose:** Quick product showcase ad  
> **Platform:** Kling AI video generation

---

## Primary Prompt (Copy This)

```
A software developer sits at a modern desk with multiple monitors showing red error messages and code warnings. The mood is tense, blue ambient lighting. 

The developer opens a sleek dark-themed web application. They drag a folder icon into a glowing circular upload zone. The interface shows "Analyzing..." with three parallel progress bars, each with a distinct icon representing different AI analyzers working simultaneously.

The progress bars fill with smooth animations. Green checkmarks appear sequentially next to each analyzer. The final screen displays a large glowing green shield icon with "VERIFIED" text and "96% Confidence" below it.

The developer leans back with a satisfied expression, arms crossed, nodding approvingly at the screen.

Style: Cinematic, modern tech aesthetic, dark UI with blue and green accent lighting, shallow depth of field, smooth slow camera movements, professional corporate feel, 4K quality.

Mood: Tension transforming to relief and confidence.
```

## Post-Production Additions

After Kling AI generates the video, add in editing software:

1. **Opening Text (0-2s):** "Your Code. Three AI Judges."
2. **End Card (13-15s):** 
   - CVA Logo
   - Tagline: "Three AIs. One Verdict. Zero Doubt."
   - URL: [your-domain]

3. **Sound Design:**
   - Subtle tension music at start
   - Satisfying "success" chime on verification
   - Modern electronic ambient track

## Brand Guidelines for Consistency

- **Primary Colors:** Dark background (#0a0a0a), Blue accent (#3b82f6), Green success (#22c55e)
- **Typography:** Modern sans-serif (Inter, SF Pro)
- **Aesthetic:** Developer-focused, professional, trustworthy
- **Mood:** Sophisticated, confident, efficient

---

## Usage Notes

1. Generate multiple variations and select the best
2. Kling AI may not render text accurately - add text in post
3. Request 4K output for quality
4. Consider generating 2-3 versions with slight prompt variations
5. Human face generation quality varies - may need to use abstract version if faces look uncanny
